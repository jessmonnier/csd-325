About This Site

This is an experimental Django site for me (Jess Monnier) to play around with Python in the context of a website. I used this YouTube video ("Learn Django in 20 Minutes!" by Tech with Tim) and this GitHub distro ("django-todo" by shreys7) as the basis for the Todo App on this site, but I made some tweaks:

  • Since the base.html template in the video tutorial uses Bootstrap 4, which doesn't support glyphicons, I had to switch the trashcan icon to SVG in a span that uses an adjusted version of the original icon's CSS styling.
  • I used the default static folder model for Django rather than the adjusted one the GitHub distro was using.
  • I added this page and updated the links in base.html to be functional.

No major changes, just little things I had to figure out to get the two slightly different templates to work together properly. I went through a couple different todo app templates before finding one that was the right fit for my current level of understanding about Django. I'm excited to grow my expertise in this area!
